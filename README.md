# Playlist de paz

Base estática para una web de disculpas personal, divertida y romántica.

## Estructura
- `index.html`: contenido principal de la página.
- `styles.css`: diseño visual, animaciones y responsive.
- `script.js`: interacción ligera del botón sorpresa.
- `render.yaml`: configuración para desplegar como sitio estático en Render.

## Cómo usar tus imágenes
Reemplaza los bloques de galería y la imagen principal con tus fotos cuando quieras.
Puedes usar estos nombres si luego agregas archivos reales:
- `assets/hero.jpg`
- `assets/photo-1.jpg`
- `assets/photo-2.jpg`
- `assets/photo-3.jpg`
- `assets/photo-4.jpg`

## Deploy en Render
1. Sube este proyecto a GitHub.
2. En Render, crea un nuevo Static Site.
3. Usa la rama principal del repositorio.
4. Publica la carpeta raíz del proyecto.
